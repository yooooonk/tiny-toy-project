## 어떻게 할까?

- 강의 내용을 블로그에 정리한다

# 요구사항 정리

- 추가
  [] 확인 버튼을 누르면 메뉴가 추가된다
  [] 엔터를 누르면 메뉴가 추가된다
  [] 메뉴가 추가되고 input이 초기화된다
  [] count가 증가한다
- 수정
  [] prompt가 뜬다
  [] 확인버튼을 누르면 수정한 내용이 목록에 반영된다
- 삭제
  [] 삭제를 묻는 confirm이 뜬다
  [] 확인을 누르면 메뉴가 삭제된다
  [] count가 감소한다
